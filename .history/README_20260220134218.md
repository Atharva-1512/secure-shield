# Secure Shield

Secure Shield is a WordPress security hardening plugin focused on:

- Login protection
- Audit logging
- Secure coding practices
- Clean OOP architecture

## Architecture

- PSR-4 Autoloading
- Namespaced classes
- Custom database table for logs
- Secure WordPress hooks implementation
